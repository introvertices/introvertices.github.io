Summary:

I completed this as part of Florinpop17's App Ideas.
This was the first intermediate project I approached. Tasks are added to an array and drawn into the list's innerHTML when added or removed. IDs are created from the array index for the buttons and items, so that I can remove them easily. I used what I had learnt from the Tic Tac Toe game to implement the close buttons (by getting all of the buttons on the list, then checking the id and splicing that index out of the task array).

I did ensure I sanitised task input via DOMPurify as well, since my Regex approach from the palindrome check wouldn't work as needed here.

I didn't find this too tough to be honest - I've certainly seen more code-complex but feature-same solutions than mine when I went to compare what I'd done. Overall happy and proud of what I made though!
